## 简介

Whale API 是面向券商提供的一套高级接口能力，券商可以选择通过自研或者采购第三方研发平台完成开发对接 Whale API 实现展业能力。

## 接口类型

Whale API 提供接入底层服务的 HTTP 接口，同时提供 WebSockets 长连接来同步订单状态。
